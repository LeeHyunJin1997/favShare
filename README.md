#  favS2hare - 유튜브 콘텐츠 기반 SNS

![로고](./docs/image/favS2hare.png)

## ✔ 소개
*- 내 최애를 골라보고 POP을 통해 자랑하고 피드를 꾸며 타인과 공유할 수 있는 SNS -*


## ✔ 주요 기술
---

**Backend - Spring**
- Eclipse 2020-03 (4.15.0)
- Spring Data JPA
- Spring Web
- Swagger 3.0.0

**Frontend**
- Visual Studio Code IDE
- Vue 2.7.8
- Vuetify 2.6.7
- Vuex 3.6.2

**CI/CD**
- Jenkins
- NGINX
